❇️语音电话小秘书是一款安卓App，功能是拨打/接听电话并静默播放语音文件，可根据用户应答播放不同的音频，带录音和转接客服电话功能，用于适量客户通知、无人值守来电管理等。
️️❇️主要功能简介如下：
1️⃣拨打电话并自动静默播放音频文件：
👉声音只能对方听到，本手机侧静音。
👉对方可根据语音提示，说出自己的应答，App通过AI识别对方应答，再播放不同的音频，类似10086的客服电话。
👉对方可选择转接客服电话，类似于10086的转人工客服。
👉可以设置多个电话，按照一定间隔拨打，失败可重拨。
👉音频文件自己定制。
👉生成详细的拨打日志，包括通话时间，应答，关机/拒接/无法接通等状态。

2️⃣自动接听来电并静默播放音频文件：
👉播放音频方面同上。
👉相当于来电管理小秘书。

3️⃣可以录音来电，去电，只录对方声音：
👉录音文件保存在本地。

4️⃣可以将来电/去电/短信等上传服务端：
👉可根据需求定制，快速交付。

️️❇️可购买安装有定制ROM的手机(推荐💪)，安卓16系统，无需root，不用安装magisk等敏感软件。
️️❇️也可自备谷歌手机(推荐6以上)+magisk，当做模块安装
