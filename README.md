<h1>App外呼系统</h1>
一个App就是一部外呼系统+智能电话客服，自动拨打/接听电话，并播放音频文件，营销利器，全网独家，用自己的卡号，不用买这买那，功能很多，超乎你想象。<br><br>
📳App必须安装在谷歌手机，配套专门定制ROM，运行时需要太多特殊权限，AI语音识别也需要大内存。<br>
📳自研ROM专门适配pixel6(oriole)及以上手机，移动/联通/电信/广电 4网5G，全网独家支持。<br><br>
❇️语音电话小秘书是一款安卓App，功能是拨打/接听电话并静默播放语音文件，可根据用户应答播放不同的音频，带录音和转接客服电话功能，用于适量客户通知、无人值守来电管理等。<br>
️️❇️主要功能简介如下：<br>
1️⃣拨打电话并自动静默播放音频文件：<br>
👉声音只能对方听到，本手机侧静音。<br>
👉对方可根据语音提示，说出自己的应答，App通过AI识别对方应答，再播放不同的音频，类似10086的客服电话。<br>
👉对方可选择转接客服电话，类似于10086的转人工客服。<br>
👉可以设置多个电话，按照一定间隔拨打，失败可重拨。<br>
👉音频文件自己定制。<br>
👉生成详细的拨打日志，包括通话时间，应答，关机/拒接/无法接通等状态。<br>

2️⃣自动接听来电并静默播放音频文件：<br>
👉播放音频方面同上。<br>
👉相当于来电管理小秘书。<br>

3️⃣可以录音来电，去电，只录对方声音：<br>
👉录音文件保存在本地。<br>

4️⃣可以将来电/去电/短信等上传服务端：<br>
👉可根据需求定制，快速交付。<br>
